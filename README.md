A curated collection of useful Shell scripts to automate tasks, manage systems, and streamline DevOps workflows. This repository serves as a personal laboratory for learning and implementing automation using Bash.

## 🚀 Overview

This repository contains various scripts ranging from basic utility tools to more complex automation workflows. Whether you are a beginner looking to understand syntax or a professional looking for a quick snippet, you'll find it here.

## 📂 Repository Structure

| Script Name | Description |
| :--- | :--- |
| `backup.sh` | Automates the backup of specified directories to a compressed archive. |
| `system_health.sh` | Monitors CPU, Memory, and Disk usage with alerts. |
| `user_management.sh` | Simplifies adding/removing users and managing permissions. |
| `network_check.sh` | Pings multiple hosts and logs connectivity status. |
| `cleanup.sh` | Clears temporary files and logs older than X days. |
